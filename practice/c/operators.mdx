## Q1. Arithmetic Operators

Demonstrate all arithmetic operators on two integers.

```c
#include <stdio.h>

int main() {
    int a = 10, b = 3;
    printf("Add: %d\n", a + b);
    printf("Sub: %d\n", a - b);
    printf("Mul: %d\n", a * b);
    printf("Div: %d\n", a / b);
    printf("Mod: %d\n", a % b);
    return 0;
}
```

## Q2. Relational Operators

Check whether two numbers are equal, greater, or lesser.

```c
#include <stdio.h>

int main() {
    int a, b;
    scanf("%d %d", &a, &b);
    printf("a == b : %d\n", a == b);
    printf("a >  b : %d\n", a >  b);
    printf("a <  b : %d\n", a <  b);
    return 0;
}
```

## Q3. Bitwise AND & OR

Print the result of bitwise AND and OR of two numbers.

```c
#include <stdio.h>

int main() {
    int a = 6, b = 3;  // 110 & 011
    printf("AND: %d\n", a & b);  // 010 = 2
    printf("OR : %d\n", a | b);  // 111 = 7
    return 0;
}
```
