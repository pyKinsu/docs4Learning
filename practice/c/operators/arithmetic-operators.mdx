## Problem Statement

Demonstrate all arithmetic operators (addition, subtraction, multiplication, division, modulus) on two integers.

## Solution

```c
#include <stdio.h>

int main() {
    int a = 10, b = 3;
    printf("Add: %d\n", a + b);
    printf("Sub: %d\n", a - b);
    printf("Mul: %d\n", a * b);
    printf("Div: %d\n", a / b);
    printf("Mod: %d\n", a % b);
    return 0;
}
```

## Output

```
Add: 13
Sub: 7
Mul: 30
Div: 3
Mod: 1
```
