## Problem

Write a C program that prints `Hello, World!` followed by a newline to standard output.

## Expected output

```
Hello, World!
```

## Solution

```c hello.c
#include <stdio.h>

int main(void) {
    printf("Hello, World!\n");
    return 0;
}
```

## How to compile and run

```bash
gcc -std=c11 -Wall -o hello hello.c
./hello
```

## Explanation

| Line | What it does |
|---|---|
| `#include <stdio.h>` | Includes the Standard I/O header, which declares `printf` |
| `int main(void)` | Entry point of every C program — execution starts here |
| `printf("Hello, World!\n")` | Writes the string to stdout; `\n` moves to the next line |
| `return 0` | Signals successful execution to the operating system |
