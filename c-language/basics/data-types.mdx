## Primitive types

C has four fundamental type categories:

| Category | Types |
|---|---|
| Integer | `char`, `short`, `int`, `long`, `long long` |
| Unsigned integer | `unsigned char`, `unsigned int`, etc. |
| Floating-point | `float`, `double`, `long double` |
| Void | `void` |

## Integer types

```c
char   c = 'A';          // 1 byte,  -128 to 127
short  s = 1000;         // 2 bytes, -32,768 to 32,767
int    i = 100000;       // 4 bytes, -2,147,483,648 to 2,147,483,647
long   l = 100000L;      // 4 or 8 bytes (platform-dependent)
long long ll = 1LL;      // 8 bytes, very large range
```

<Info>
  The exact size of `int` and `long` depends on your platform and compiler. Use `sizeof(int)` to check at runtime. For guaranteed sizes, use `<stdint.h>` types like `int32_t`.
</Info>

## Fixed-width integers (preferred in modern C)

```c
#include <stdint.h>

int8_t   a = -5;           // exactly 8 bits, signed
uint8_t  b = 255;          // exactly 8 bits, unsigned
int32_t  c = 100000;       // exactly 32 bits, signed
uint64_t d = 1000000000ULL; // exactly 64 bits, unsigned
```

Use these when the exact size matters — network protocols, file formats, embedded systems.

## Floating-point types

```c
float  f = 3.14f;          // 4 bytes, ~6-7 significant digits
double d = 3.14159265358;  // 8 bytes, ~15-16 significant digits
```

<Warning>
  Floating-point numbers are **not exact**. `0.1 + 0.2` does not equal `0.3` in binary floating-point. Never use `==` to compare floats — use a small epsilon instead.
</Warning>

```c
// Wrong
if (a == b) { ... }

// Correct
if (fabs(a - b) < 1e-9) { ... }
```

## The `char` type

`char` stores a single character, but internally it is just a small integer. Characters are mapped via the ASCII table:

```c
char letter = 'A';
printf("%c\n", letter);     // prints: A
printf("%d\n", letter);     // prints: 65  (ASCII value of 'A')
```

## Boolean (C99+)

C did not have a native boolean type until C99. Include `<stdbool.h>`:

```c
#include <stdbool.h>

bool is_valid = true;
bool is_empty = false;
```

Without `<stdbool.h>`, you can use `int` where `0` is false and any non-zero value is true.

## Type conversion

C performs **implicit conversion** (coercion) when mixing types. Go from narrower to wider to avoid data loss:

```c
int   i = 10;
float f = i;       // implicit: int → float, safe

float g = 3.99f;
int   j = g;       // implicit: float → int, truncates decimal (j = 3)
```

Use **explicit casting** when you intend to convert:

```c
int a = 7, b = 2;
float result = (float)a / b;   // 3.5 — without cast, result would be 3
```

## Checking sizes with `sizeof`

```c
#include <stdio.h>

int main(void) {
    printf("char:      %zu bytes\n", sizeof(char));
    printf("int:       %zu bytes\n", sizeof(int));
    printf("long:      %zu bytes\n", sizeof(long));
    printf("double:    %zu bytes\n", sizeof(double));
    return 0;
}
```
