## Output with `printf`

`printf` (print formatted) writes formatted text to standard output.

```c
#include <stdio.h>

int main(void) {
    int age = 21;
    float gpa = 3.85f;
    char name[] = "Kinsu";

    printf("Name: %s\n", name);
    printf("Age:  %d\n", age);
    printf("GPA:  %.2f\n", gpa);  // 2 decimal places
    return 0;
}
```

## Format specifiers

| Specifier | Type | Example |
|---|---|---|
| `%d` or `%i` | `int` | `42` |
| `%u` | `unsigned int` | `42` |
| `%f` | `float` / `double` | `3.14` |
| `%.2f` | float with 2 decimals | `3.14` |
| `%e` | scientific notation | `3.14e+00` |
| `%c` | `char` | `A` |
| `%s` | string (`char*`) | `hello` |
| `%p` | pointer address | `0x7fff...` |
| `%x` | hexadecimal | `1a` |
| `%o` | octal | `52` |
| `%zu` | `size_t` (sizeof results) | `8` |
| `%%` | literal `%` | `%` |

## Width and padding

```c
printf("%10d\n", 42);      //         42  (right-aligned, 10 wide)
printf("%-10d|\n", 42);    // 42        | (left-aligned)
printf("%010d\n", 42);     // 0000000042 (zero-padded)
printf("%+d\n", 42);       // +42        (show sign)
```

## Input with `scanf`

`scanf` reads formatted input from standard input (keyboard).

```c
int age;
printf("Enter your age: ");
scanf("%d", &age);   // & passes the address of age
printf("You entered: %d\n", age);
```

<Warning>
  The `&` (address-of) operator is **required** for most types in `scanf`. Forgetting it is one of the most common bugs in C — it causes undefined behavior or a crash.
</Warning>

## Reading multiple values

```c
int x, y;
scanf("%d %d", &x, &y);   // reads two integers separated by whitespace
```

## Reading strings safely

```c
char name[50];
scanf("%49s", name);        // reads up to 49 chars, no & needed for arrays
```

<Warning>
  Never use `scanf("%s", name)` without a width limit. It does not check the buffer size and causes a **buffer overflow** — a critical security vulnerability.
</Warning>

## `fgets` — safer string input

For reading full lines including spaces, prefer `fgets` over `scanf`:

```c
char line[100];
fgets(line, sizeof(line), stdin);
// fgets includes the trailing newline — strip it if needed:
line[strcspn(line, "\n")] = '\0';
```

|---|---|
