## Basic syntax

`switch` tests a single integer expression against multiple constant values:

```c
int day = 3;

switch (day) {
    case 1:
        printf("Monday\n");
        break;
    case 2:
        printf("Tuesday\n");
        break;
    case 3:
        printf("Wednesday\n");
        break;
    default:
        printf("Other day\n");
        break;
}
```

## The `break` statement

Without `break`, execution **falls through** to the next case. This is a common source of bugs:

```c
switch (x) {
    case 1:
        printf("one\n");
        // no break — falls through!
    case 2:
        printf("two\n");
        break;
}
// If x == 1, prints both "one" and "two"
```

Always add `break` unless you intentionally want fall-through. When fall-through is intentional, add a comment:

```c
case 1:
    // fall through intentional
case 2:
    printf("one or two\n");
    break;
```

## The `default` case

Matches when no `case` matches. It can appear anywhere, but conventionally goes last:

```c
switch (ch) {
    case 'a': case 'e': case 'i': case 'o': case 'u':
        printf("vowel\n");
        break;
    default:
        printf("consonant\n");
        break;
}
```

## Grouping cases

Multiple `case` labels can share one block — a clean way to handle alternatives:

```c
switch (month) {
    case 1: case 3: case 5: case 7:
    case 8: case 10: case 12:
        days = 31;
        break;
    case 4: case 6: case 9: case 11:
        days = 30;
        break;
    case 2:
        days = 28;  // simplified, ignoring leap years
        break;
}
```

## `switch` vs `if-else`

| | `switch` | `if-else` |
|---|---|---|
| Tests | Single integer/char expression | Any expression |
| Values | Compile-time constants only | Variables, ranges, functions |
| Fall-through | Yes (explicit `break` needed) | No |
| Readability | Cleaner for many fixed values | More flexible |

Use `switch` when you're comparing one value against many fixed integer or character constants. Use `if-else` for ranges (`x > 10`), floats, or string comparisons.

<Info>
  `switch` works with `int`, `char`, `short`, `long`, and `enum` types. It does not work with `float`, `double`, or strings.
</Info>
