## The `for` loop

Best when the number of iterations is known upfront:

```c
for (int i = 0; i < 5; i++) {
    printf("%d\n", i);
}
// prints 0, 1, 2, 3, 4
```

The three parts of a `for` loop:

```
for (initializer; condition; update) { body }
```

- **Initializer** — runs once before the loop starts
- **Condition** — checked before each iteration; loop stops when false
- **Update** — runs after each iteration

Any part can be omitted:

```c
// Infinite loop
for (;;) {
    // runs forever — use break to exit
}
```

## The `while` loop

Best when the number of iterations is unknown — keep looping while a condition holds:

```c
int count = 0;
while (count < 5) {
    printf("%d\n", count);
    count++;
}
```

If the condition is false from the start, the body never runs.

## The `do-while` loop

Like `while`, but the body always executes **at least once** — the condition is checked after:

```c
int input;
do {
    printf("Enter a positive number: ");
    scanf("%d", &input);
} while (input <= 0);
```

This is the natural fit for input validation loops.

## `break` and `continue`

```c
for (int i = 0; i < 10; i++) {
    if (i == 3) continue;  // skip 3
    if (i == 7) break;     // stop at 7
    printf("%d ", i);
}
// prints: 0 1 2 4 5 6
```

- `break` — exits the **entire** loop immediately
- `continue` — skips the rest of the current iteration, jumps to the update step

## Nested loops

```c
for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 3; j++) {
        printf("%d×%d=%d  ", i, j, i * j);
    }
    printf("\n");
}
```

`break` inside a nested loop only exits the **inner** loop. To exit multiple levels, use a flag variable or restructure with a function.

## Common loop patterns

**Counting down:**
```c
for (int i = 10; i > 0; i--) {
    printf("%d\n", i);
}
```

**Iterating over an array:**
```c
int arr[] = {10, 20, 30, 40, 50};
int len = sizeof(arr) / sizeof(arr[0]);

for (int i = 0; i < len; i++) {
    printf("%d\n", arr[i]);
}
```

**Reading until EOF:**
```c
int ch;
while ((ch = getchar()) != EOF) {
    putchar(ch);
}
```
