## What is recursion?

A function is **recursive** if it calls itself. Every recursive solution has two parts:

1. **Base case** — a condition that stops the recursion
2. **Recursive case** — a call to itself with a smaller/simpler input

## Classic example: factorial

```c
// n! = n × (n-1) × (n-2) × ... × 1
// 0! = 1  (base case)

int factorial(int n) {
    if (n <= 0) return 1;         // base case
    return n * factorial(n - 1);  // recursive case
}

int main(void) {
    printf("%d\n", factorial(5));  // 120
    return 0;
}
```

**Call trace for `factorial(4)`:**
```
factorial(4)
  → 4 * factorial(3)
       → 3 * factorial(2)
            → 2 * factorial(1)
                 → 1 * factorial(0)
                      → 1   (base case)
```

## Fibonacci sequence

```c
int fib(int n) {
    if (n <= 1) return n;
    return fib(n - 1) + fib(n - 2);
}
```

<Warning>
  This naive Fibonacci is **exponentially slow** — `fib(50)` makes billions of calls. For performance, use an iterative approach or memoization.
</Warning>

## Recursive vs iterative

Most recursive solutions have an equivalent iterative version. The right choice depends on clarity and constraints:

| | Recursive | Iterative |
|---|---|---|
| Readability | Often cleaner for tree/graph problems | Cleaner for simple loops |
| Stack usage | Uses call stack (risk of overflow) | Uses constant stack space |
| Performance | Function call overhead | Generally faster |
| Use case | Trees, divide-and-conquer, backtracking | Counting, accumulation |

## Stack overflow

Each recursive call uses stack space. Too many levels deep causes a **stack overflow**:

```c
void infinite(void) {
    infinite();  // no base case — crashes
}
```

Default stack sizes are typically 1–8 MB. Deep recursion (thousands of levels) should be refactored to iterative or use explicit stacks.

## Practical example: binary search (recursive)

```c
int binary_search(int arr[], int low, int high, int target) {
    if (low > high) return -1;  // not found

    int mid = low + (high - low) / 2;

    if (arr[mid] == target) return mid;
    if (arr[mid] < target) return binary_search(arr, mid + 1, high, target);
    return binary_search(arr, low, mid - 1, target);
}
```
