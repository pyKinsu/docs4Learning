## Why functions?

Functions let you break a program into named, reusable pieces. A well-structured C program is mostly a collection of small, focused functions.

## Anatomy of a function

```c
return_type function_name(parameter_type parameter_name, ...) {
    // body
    return value;
}
```

**Example:**

```c
int add(int a, int b) {
    return a + b;
}

int main(void) {
    int result = add(3, 4);
    printf("%d\n", result);  // 7
    return 0;
}
```

## Void functions

Functions that perform an action but return no value use `void` as the return type:

```c
void print_separator(void) {
    printf("-------------------\n");
}
```

## Function declarations (prototypes)

In C, a function must be **declared** before it is used. A prototype tells the compiler the function's signature without providing the body:

```c
// Prototype at the top of the file
int multiply(int a, int b);

int main(void) {
    printf("%d\n", multiply(6, 7));  // 42
    return 0;
}

// Definition can come after main
int multiply(int a, int b) {
    return a * b;
}
```

<Tip>
  In larger projects, prototypes go in **header files** (`.h`), and definitions go in **source files** (`.c`). This is how the C standard library works.
</Tip>

## Parameters are passed by value

C passes arguments **by value** — functions receive copies, not the originals:

```c
void double_it(int x) {
    x = x * 2;  // only modifies the local copy
}

int main(void) {
    int n = 5;
    double_it(n);
    printf("%d\n", n);  // still 5
    return 0;
}
```

To modify the caller's variable, pass a **pointer** instead. See the [Pointers](/c-language/pointers/introduction) section.

## Multiple return values (via pointers)

C functions return only one value, but you can simulate multiple returns with pointer parameters:

```c
void min_max(int arr[], int len, int *min, int *max) {
    *min = *max = arr[0];
    for (int i = 1; i < len; i++) {
        if (arr[i] < *min) *min = arr[i];
        if (arr[i] > *max) *max = arr[i];
    }
}

int main(void) {
    int nums[] = {3, 1, 8, 2, 5};
    int lo, hi;
    min_max(nums, 5, &lo, &hi);
    printf("Min: %d, Max: %d\n", lo, hi);  // Min: 1, Max: 8
    return 0;
}
```

## Static functions

A function declared `static` is only visible within the file it is defined in — it cannot be called from other translation units:

```c
static int helper(int x) {
    return x * x;
}
```

Use `static` for internal helper functions to prevent name collisions across files.
