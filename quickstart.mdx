## Contribute

This documentation is open source. To suggest a fix or add an example:

1. Fork the [GitHub repository](https://github.com/pyKinsu/docs)
2. Create a branch for your change
3. Submit a pull request with a clear description

See [CONTRIBUTING.md](https://github.com/pyKinsu/docs/blob/main/CONTRIBUTING.md) for the full contribution guide.
